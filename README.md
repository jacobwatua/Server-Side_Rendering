#Instructions

<p>Copy main.py into a directory of your choice</p>
<p>Run:</p>
<code>
    python main.py
</code>

<p>Navigate to the project folder created and run:</p>
<code>
    npm run start
        OR
    nodemon (If you have nodemon installed globally)
</code>
